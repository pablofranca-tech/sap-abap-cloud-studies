# ABAP Cloud Learning

Repositório com exercícios e exemplos desenvolvidos durante meus estudos de ABAP Cloud utilizando SAP BTP Trial e Eclipse ADT.

## Objetivo

Aprender os fundamentos do desenvolvimento ABAP Cloud, incluindo:

- Classes ABAP
- Interfaces
- Tipos de dados
- Variáveis
- Constantes
- Programação Orientada a Objetos
- Banco de Dados SAP HANA
- RAP (RESTful ABAP Programming Model)

## Ambiente Utilizado

- SAP BTP Trial
- ABAP Environment (ABAP Cloud)
- Eclipse IDE
- ABAP Development Tools (ADT)

## Exercícios

### 1. Hello World

Arquivo:

```text
zcl_pablo_hello_world.abap
```

Conceitos praticados:

- Criação de classes
- Interface IF_OO_ADT_CLASSRUN
- Saída no console com OUT->WRITE

### 2. Data Types

Arquivo:

```text
zcl_pablo_data_types.abap
```

Conceitos praticados:

- Tipos primitivos ABAP
- STRING
- INTEGER (I)
- DATE (D)
- CHAR (C)
- NUMERIC TEXT (N)
- PACKED NUMBER (P)

### 3. Variables and Constants

Arquivo:

```text
zcl_pablo_variables.abap
```

Conceitos praticados:

- Variáveis
- Tipos locais (TYPES)
- Tipos globais SAP
- Constantes
- Literais
